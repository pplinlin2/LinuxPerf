# CPU
## Utilization
### System Wide
`vmstat 1`中cpu欄位的us、sy、wa總和
* us: user time
* sy: system time
* wa: waiting time for IO

`mpstat`欄位中除了%idle和%iowait之外的總和

## Saturation
### System Wide
`vmstat 1`當中r > CPU數量
* r: #runnable process，包括running和waiting for run time
* b: #uninterruptible process
